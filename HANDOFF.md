# HANDOFF — Mopiko clean (для нового чату / іншого агента / розробника)

Прочитай цей файл **першим**. Він дає повний контекст, щоб не перечитувати весь код.
Сайт **у продакшні**: https://mopiko.pl

## Що це
Сайт клінінгової компанії **Mopiko clean** (Познань, Польща).
Стек: **статичний HTML + CSS + vanilla JS, без збірки (no build step)**.
Двомовний: **польська — основна** (`/`, `<html lang="pl">`), **українська** (`/ua/`).
Основна аудиторія — польський ринок.

## Хостинг і деплой
- **Vercel** — автодеплой з гілки `main` GitHub-репозиторію. Пуш у `main` → сайт оновлюється сам.
- **Домен**: `mopiko.pl` — реєстратор GoDaddy, A-запис apex указує на Vercel. HTTPS активний.
- **Репозиторій публічний** → **нічого секретного в код не комітити**. Усі секрети — в Environment Variables проєкту Vercel.
- **Cache-busting**: браузери агресивно кешують. Змінюючи CSS/JS — підіймай `?v=N` у `<link>/<script>` на всіх сторінках, що його підключають.
- Локальний запуск: `python3 -m http.server 4180` → http://localhost:4180/

## JS (усі у `js/`)
- `config.js` — **КОНТАКТИ Й НАЛАШТУВАННЯ** (`window.MOPIKO`): phone, telegram, whatsapp, email, promoCodes, leadEndpoint (`/api/lead`). Один файл — усі сторінки.
- `consent-init.js` — вантажиться в `<head>`: Google **Consent Mode v2** (defaults denied) + завантаження **GTM `GTM-5DQ5J2GW`**.
- `notice.js` — **cookie-банер** із гранульованими налаштуваннями (прийняти все / тільки необхідні / налаштування), пише вибір у localStorage `mopiko-consent`, оновлює consent через gtag. Лінк «налаштування» у футері.
- `nav.js` — телефон/telegram у розмітці, перемикач мови; інжектить **JSON-LD LocalBusiness/CleaningService**.
- `order.js` — уся логіка сторінки замовлення (калькулятор, валідація, повідомлення, заявка).
- `home.js` — головна (hero-калькулятор, ціни, галерея до/після, відгуки, GSAP); інжектить **JSON-LD FAQPage**.
- `what.js` — інтерактивні кімнати «Що входить».
- `fx.js` — магнітні кнопки (`.magnet`).

## CSS
- `css/styles.css` — **дизайн-токени в `:root`** + спільні стилі (кнопки, форми, футер, cookie, сторінка замовлення).
- `css/home.css` — стилі головної.
- `css/fonts.css` — self-hosted шрифти (`/fonts`, розбиті по subset). **Unbounded** (заголовки) + **Golos Text** (текст), усі `font-display: swap`.
- Глобальна зміна кольору/шрифту/радіуса — одна правка в `:root`.

## Аналітика і згода (важливо)
- **GTM**: `GTM-5DQ5J2GW`, вантажиться з `consent-init.js`. GA4 підключена **через GTM** (не хардкодом).
- **Consent Mode v2 (advanced)**: до згоди відвідувача analytics/ads = `denied`, куки не ставляться (cookieless pings). Готово під майбутні **Meta Pixel + Google Ads** — додавати теги в GTM, вони автоматично слухають consent.
- Акаунти GTM / GA4 / Search Console — під Google-акаунтом клієнта.

## SEO
- JSON-LD: LocalBusiness/CleaningService (`nav.js`) + FAQPage (`home.js`).
- hreflang `pl`/`uk`/`x-default`, canonical, OG/Twitter (`img/og-pl.png`, `img/og-uk.png`), `sitemap.xml`, `robots.txt`.
- Title/description — під пошукові запити (keyword-first). Запитано індексацію в Google Search Console.

## Заявки з форми → Telegram
- Форма «залишити заявку» → `POST /api/lead` (`api/lead.js`, Vercel serverless).
- **Env-змінні у Vercel** (обов'язкові): `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID` (кілька одержувачів — через кому).
- Бот створений у BotFather під робочим Telegram-акаунтом `@mopiko_pl`; заявки приходять туди. Перевірено — працює.
- Кнопки «написати в Telegram/WhatsApp» ведуть на контакти з `config.js`. Захист ендпоінта: перевірка Origin == Host.

## Бізнес-логіка калькулятора (order.js + дубль у home.js)
- Ціна: `95 (виїзд+кухня+коридор) + кімнати*40 + санвузли*50` (1+1=185, 2+1=225). Будинок `×1.2`. Кухня-студія `−10`. Після ремонту — `6 zł/м²`, вікна в цьому режимі `+20%`.
- Опції залежать від типу: балкони — лише квартира; сходи/ґанок/тераса — лише будинок.
- Список опцій згорнутий до топ-8 + «дивитись усі».
- Офіс — `5 zł/м²`, **не в калькуляторі**: ціна й CTA у секції «Для бізнесу».
- Періодичність: разове (0%) → місяць (−10%) → два тижні (−15%) → тиждень (−20%). Мінімум **3 години**.
- Міні-калькулятор на головній (`home.js`) дублює формулу — **міняючи модель, правити обидва файли**.
