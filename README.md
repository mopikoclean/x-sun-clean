# x-sun clean — сайт клінінгової компанії (Познань)

Статичний сайт без збірки: HTML + CSS + vanilla JS.

## Сторінки

- [index.html](index.html) — головна сторінка: hero з міні-калькулятором, послуги (горизонтальний скрол із пінінгом на GSAP), ціни з періодичністю та доп. опціями, слайдер «до/після», процес, зона обслуговування, блок для бізнесу, FAQ, футер, плаваюча кнопка зв'язку (за дизайном `design/project/X-sun Clean.dc.html`)
- [zamovlennya.html](zamovlennya.html) — сторінка замовлення з калькулятором вартості (за дизайном `design/project/Замовлення.dc.html`)
- [shcho-prybyraiemo.html](shcho-prybyraiemo.html) — «Що входить у прибирання»: інтерактивні ізометричні кімнати з хотспотами (дані точок — у `js/what.js`, зображення — `img/room-*.webp` з прозорим фоном)
- [privacy.html](privacy.html) — політика конфіденційності (GDPR/RODO)

## Юридичне (GDPR)

- **Політику конфіденційності** ([privacy.html](privacy.html)) потрібно **перевірити юристом** і заповнити реальні реквізити компанії (назва, адреса, NIP/REGON) — у тексті позначені як `[вкажіть …]`.
- **Банер cookie** з'являється при першому візиті: затемнює сторінку й блокує прокрутку, доки користувач не натисне «Прийняти» або «Відхилити». Аналітику підключати лише у `enableAnalytics()` в `js/cookie.js` — вона має вантажитися **тільки після** згоди.

## Перед публікацією

Замініть контакти у `js/config.js` (використовуються обома сторінками):

- `phone` — реальний номер телефону (зараз `+48 XXX XXX XXX`)
- `telegram` — нік Telegram-акаунта (зараз `xsunclean`)
- `whatsapp` — номер WhatsApp без `+` (зараз `48000000000`)
- `email` — контактна пошта у футері (зараз `info@xsunclean.pl`)
- `promoCodes` — словник промокодів `КОД: частка знижки` (зараз тестовий `START15: 0.15`)
- `leadEndpoint` — ендпоінт для заявок «ми передзвонимо» (кнопка на сторінці замовлення для клієнтів без месенджерів). Найпростіше: створіть безкоштовну форму на [formspree.io](https://formspree.io), вкажіть пошту менеджера і вставте URL форми (`https://formspree.io/f/…`). Поки поле порожнє, кнопка чесно просить зателефонувати.

Відгуки на головній (`index.html`, секція `#vidguky`) — зараз демонстраційні; замініть на реальні, щойно з'являться.

## Дизайн-система

**Живий гайд:** [dyzajn-systema.html](dyzajn-systema.html) — усі кольори, шрифти, кнопки, поля, чипи, радіуси й тіні на одній сторінці (намальовані тими самими класами, що й сайт).

**Токени** — єдине джерело істини у `:root` в `css/styles.css`:
- **Кольори** — поверхні (`--bg`, `--surface`, `--surface-warm`), текст (`--ink`, `--ink-soft`, `--muted`, `--muted-light`), межі (`--border`, `--border-strong`), бренд (`--accent`, `--accent-dark`, `--gold-text`, `--badge-bg`), статуси (`--danger`, `--danger-soft`).
- **Шрифти** — `--font-display` (Unbounded, заголовки) + `--font-body` (Golos Text, текст).
- **Радіуси** — `--radius-sm/md/…/2xl` + `--radius-pill`.
- **Тіні** — `--shadow-pop/card/lift/fab/modal/menu`.
- **Рух** — `--ease` (фірмовий easing).

Уся верстка прив'язана до цих токенів — щоб змінити акцент, шрифт чи радіус на всьому сайті, достатньо однієї правки у `:root`.

**Кнопки** (`css/styles.css`, секція «Дизайн-система: кнопки») — завжди `.btn` + один клас ієрархії + опційно розмір:

| Клас | Роль | Вигляд |
|---|---|---|
| `.btn-primary` | головна дія (замовити, зателефонувати) | жовта, текст чорний, вага 700 |
| `.btn-secondary` | друга за вагою дія | чорна, текст білий |
| `.btn-ghost` | третинна на світлому фоні | контурна, рамка сіра |
| `.btn-ghost-inv` | третинна на темному фоні | контурна, рамка/текст білі |

Розміри: `.btn-sm` (35px — навігація) / базовий (46px) / `.btn-lg` (55px — футерний CTA). Модифікатори: `.btn-block` (на всю ширину), `.disabled`, `.magnet` (магнітний ефект курсора, `js/fx.js`).
