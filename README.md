# Mopiko clean — сайт клінінгової компанії (Познань)

Статичний сайт без збірки: HTML + CSS + vanilla JS. **У продакшні:** https://mopiko.pl

> **Новий тут?** Спершу прочитай [HANDOFF.md](HANDOFF.md) — повний контекст проєкту (хостинг, деплой, аналітика, заявки, відкриті пункти). Цей README — про сторінки й дизайн-систему.

## Сторінки

Двомовний сайт: польська — основна (`/`), українська (`/ua/`).

- [index.html](index.html) / [ua/index.html](ua/index.html) — головна: hero з міні-калькулятором, послуги (горизонт. скрол GSAP), ціни, «до/після», процес, зона, блок для бізнесу, відгуки, FAQ, SEO-секція, футер
- [zamowienie.html](zamowienie.html) / [ua/zamovlennya.html](ua/zamovlennya.html) — замовлення з калькулятором вартості
- [co-sprzatamy.html](co-sprzatamy.html) / [ua/shcho-prybyraiemo.html](ua/shcho-prybyraiemo.html) — «Що входить у прибирання» (інтерактивні кімнати, дані — `js/what.js`)
- [polityka-prywatnosci.html](polityka-prywatnosci.html) / [ua/privacy.html](ua/privacy.html) — політика приватності (RODO/GDPR)
- [404.html](404.html) — сторінка 404

## Юридичне (GDPR) і згода

- **Політику приватності** потрібно **перевірити юристом** і заповнити реальні реквізити (назва, адреса, NIP/REGON — плейсхолдери в тексті).
- **Cookie-банер** (`js/notice.js`) — гранульований: «прийняти все / тільки необхідні / налаштування». Працює через **Google Consent Mode v2** (`js/consent-init.js`): до згоди analytics/ads = denied, куки не ставляться. Змінити вибір — лінк «налаштування» у футері.

## Перед змінами контактів

Контакти й налаштування — у `js/config.js` (використовуються всіма сторінками):

- `phone` — номер телефону (зараз `+48 452 255 364`)
- `telegram` — нік Telegram (зараз `mopiko_pl`)
- `whatsapp` — номер WhatsApp без `+` (зараз `48452255364`)
- `email` — контактна пошта (зараз `xsunclen@gmail.com` — ⚠ перевірити)
- `promoCodes` — промокоди `КОД: частка знижки` (зараз `START15: 0.15` — ⚠ тестовий)
- `leadEndpoint` — `/api/lead` (заявки → Telegram, потребує env-змінних у Vercel, див. [HANDOFF.md](HANDOFF.md))

Відгуки на головній — демонстраційні; замінити на реальні, щойно з'являться.

## Дизайн-система

**Живий гайд:** [dyzajn-systema.html](dyzajn-systema.html) — усі кольори, шрифти, кнопки, поля, чипи, радіуси й тіні на одній сторінці (намальовані тими самими класами, що й сайт).

**Токени** — єдине джерело істини у `:root` в `css/styles.css`:
- **Кольори** — поверхні (`--bg`, `--surface`, `--surface-warm`), текст (`--ink`, `--ink-soft`, `--muted`, `--muted-light`), межі (`--border`, `--border-strong`), бренд (`--accent`, `--accent-dark`, `--gold-text`, `--badge-bg`), статуси (`--danger`, `--danger-soft`).
- **Шрифти** — `--font-display` (Unbounded, заголовки) + `--font-body` (Golos Text, текст).
- **Радіуси** — `--radius-sm/md/…/2xl` + `--radius-pill`.
- **Тіні** — `--shadow-pop/card/lift/fab/modal/menu`.
- **Рух** — `--ease` (фірмовий easing).

Уся верстка прив'язана до цих токенів — щоб змінити акцент, шрифт чи радіус на всьому сайті, достатньо однієї правки у `:root`.

**Кнопки** (`css/styles.css`, секція «Дизайн-система: кнопки») — завжди `.btn` + один клас ієрархії + опційно розмір:

| Клас | Роль | Вигляд |
|---|---|---|
| `.btn-primary` | головна дія (замовити, зателефонувати) | жовта, текст чорний, вага 700 |
| `.btn-secondary` | друга за вагою дія | чорна, текст білий |
| `.btn-ghost` | третинна на світлому фоні | контурна, рамка сіра |
| `.btn-ghost-inv` | третинна на темному фоні | контурна, рамка/текст білі |

Розміри: `.btn-sm` (35px — навігація) / базовий (46px) / `.btn-lg` (55px — футерний CTA). Модифікатори: `.btn-block` (на всю ширину), `.disabled`, `.magnet` (магнітний ефект курсора, `js/fx.js`).
